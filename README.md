# SCB Slip Checker API (Educational Purpose Only)

This project is a simple example of how to integrate and use an API to verify payment slips via  
:contentReference[oaicite:0]{index=0} (SCB).

> ⚠️ This project is created **for educational purposes only**  
> It must NOT be used for any commercial, production, or unauthorized activities.

---

## 📌 Features

- Verify bank transfer slip information
- Connect to SCB Slip Verification API
- Display transaction details such as:
  - Sender / Receiver name
  - Transfer amount
  - Date and time
  - Reference ID
- Example implementation using Python
